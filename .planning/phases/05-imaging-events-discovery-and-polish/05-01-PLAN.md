    Step 2 — Create src/service/events.rs:
    - Copy extract_local_name from device.rs verbatim.

    - SubscriptionInfo struct:
      ```rust
      struct SubscriptionInfo {
          termination_time: chrono::DateTime<chrono::Utc>,
      }
      ```

    - EventServiceHandler struct:
      ```rust
      pub struct EventServiceHandler {
          pub(crate) svc: Arc<dyn EventService>,
          pub(crate) xaddr: String, // e.g. "http://0.0.0.0:8080/onvif/events_service"
          subscriptions: Arc<Mutex<HashMap<String, SubscriptionInfo>>>,
      }
      impl EventServiceHandler {
          pub fn new(svc: Arc<dyn EventService>, xaddr: impl Into<String>) -> Self {
              Self { svc, xaddr: xaddr.into(), subscriptions: Arc::new(Mutex::new(HashMap::new())) }
          }
      }
      ```

    - Dispatch:
      ```
      "GetEventProperties" => static XML response (handler-internal, not trait)
      "CreatePullPointSubscription" => self.handle_create_pull_point_subscription().await
      "PullMessages" => self.handle_pull_messages(&body).await
      "Unsubscribe" => self.handle_unsubscribe(&body).await
      _ => Err(OnvifError::ActionNotSupported.into_soap_fault())
      ```

    - handle_get_event_properties: Returns static XML:
      ```xml
      <tev:GetEventPropertiesResponse
          xmlns:tev="http://www.onvif.org/ver10/events/wsdl"
          xmlns:wsnt="http://docs.oasis-open.org/wsn/b-2"
          xmlns:wstop="http://docs.oasis-open.org/wsn/t-1">
        <tev:TopicNamespaceLocation>http://www.onvif.org/onvif/ver10/topics/topicns.xml</tev:TopicNamespaceLocation>
        <wsnt:FixedTopicSet>true</wsnt:FixedTopicSet>
        <wstop:TopicSet/>
        <wsnt:TopicExpressionDialect>http://docs.oasis-open.org/wsn/t-1/TopicExpression/Concrete</wsnt:TopicExpressionDialect>
        <wsnt:TopicExpressionDialect>http://www.onvif.org/ver10/tev/topicExpression/ConcreteSet</wsnt:TopicExpressionDialect>
        <tev:MessageContentFilterDialect>http://www.onvif.org/ver10/tev/messageContentFilter/ItemFilter</tev:MessageContentFilterDialect>
      </tev:GetEventPropertiesResponse>
      ```

    - handle_create_pull_point_subscription:
      1. sub_id = uuid::Uuid::new_v4().to_string()
      2. now = chrono::Utc::now(); termination = now + chrono::Duration::seconds(60)
      3. Insert SubscriptionInfo { termination_time: termination } into self.subscriptions with key sub_id.clone()
      4. Return XML:
         ```xml
         <tev:CreatePullPointSubscriptionResponse
             xmlns:tev="http://www.onvif.org/ver10/events/wsdl"
             xmlns:wsa5="http://www.w3.org/2005/08/addressing"
             xmlns:wsnt="http://docs.oasis-open.org/wsn/b-2">
           <tev:SubscriptionReference>
             <wsa5:Address>{xaddr}</wsa5:Address>
             <wsa5:ReferenceParameters>
               <tev:SubscriptionId>{sub_id}</tev:SubscriptionId>
             </wsa5:ReferenceParameters>
           </tev:SubscriptionReference>
           <wsnt:CurrentTime>{now.to_rfc3339()}</wsnt:CurrentTime>
           <wsnt:TerminationTime>{termination.to_rfc3339()}</wsnt:TerminationTime>
         </tev:CreatePullPointSubscriptionResponse>
         ```

    - handle_pull_messages:
      1. Extract SubscriptionId text from body using quick_xml (look for local name "SubscriptionId").
      2. Look up in subscriptions map. If not found, return SoapFault::sender("Unknown subscription").
      3. now = chrono::Utc::now(); termination = existing termination_time from map entry.
      4. Return XML:
         ```xml
         <tev:PullMessagesResponse
             xmlns:tev="http://www.onvif.org/ver10/events/wsdl"
             xmlns:wsnt="http://docs.oasis-open.org/wsn/b-2">
           <tev:CurrentTime>{now.to_rfc3339()}</tev:CurrentTime>
           <tev:TerminationTime>{termination.to_rfc3339()}</tev:TerminationTime>
         </tev:PullMessagesResponse>
         ```

    - handle_unsubscribe:
      1. Extract SubscriptionId text from body.
      2. Remove from subscriptions map (ignore if not found — idempotent).
      3. Return XML: `<tev:UnsubscribeResponse xmlns:tev="http://www.onvif.org/ver10/events/wsdl"/>`

    Step 3 — Update src/service/mod.rs to add `pub mod events;`.

    Step 4 — Write tests/events_service.rs. Use a minimal struct implementing EventService with empty get_event_properties default. Build request body bytes for each operation. For PullMessages/Unsubscribe, first call CreatePullPointSubscription to get a sub_id, then extract it from the response XML and include it in subsequent request bodies.

    Imports needed: std::collections::HashMap, std::sync::{Arc, Mutex}, uuid, chrono.
